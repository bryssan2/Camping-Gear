# Camping Gear 

Site vitrine responsive pour une marque fictive d'équipement de camping.
Projet d'entraînement personnel axé sur la mise en page multi-sections et l'intégration d'images produits.

lien du site : camping-gear-kohl.vercel.app/

##  Ce que contient le projet
- Section hero avec visuel immersif
- Présentation de la gamme de produits avec images
- Design responsive adapté mobile et desktop
- Navigation fluide entre les sections

## Technologies utilisées
- HTML5
- CSS3 / SCSS
- JavaScript (Vanilla)
- FontAwesome 6

## Ce que j'ai appris
- Intégration et optimisation d'images dans une mise en page web
- Organisation d'un projet multi-fichiers (HTML, CSS, JS, assets)
- Techniques de responsive design sans framework CSS
